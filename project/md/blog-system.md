# 个人博客系统
> 一个基于 React + Node.js 构建的全栈博客平台

## ✨ 项目亮点

- 支持 Markdown 写作
- 响应式设计
- GitHub Pages 自动部署

## 🧱 技术栈

- 前端：React, Tailwind CSS
- 后端：Node.js, Express, MongoDB
